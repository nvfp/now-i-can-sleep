# Changelog

- 2.0.0 (June 30, 2023):
    - Added `upgrade` command
    - Changed docs-tree naming format:
        - From `(<order> -- )?<name> -- <url>(.md)?` to `(<order> - )?<name>(.md)?`
        - The URL will be the lowercase and URL-safe version of the 'name'
    - Added `lowercase_the_url` option to settings
